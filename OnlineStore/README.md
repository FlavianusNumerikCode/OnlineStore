# Online Store Website

A comprehensive e-commerce platform for an online book store with full features including user authentication, product management, shopping cart, payment processing, admin dashboard, and more.

## Tech Stack
- **Frontend**: ReactJS with Tailwind CSS for responsive and interactive UI
- **Backend**: NodeJS with Express.js
- **Database**: MongoDB
- **Authentication**: JWT
- **Payment**: Stripe (mock implementation)

## Features
- User registration and login/logout
- Product catalog (books) with search, filter, and categories
- Shopping cart and wishlist
- Order management and payment
- Admin dashboard for managing products, orders, users
- User reviews and ratings
- Responsive design with animations
- Real-time notifications (using Socket.io)
- Chat support
- Analytics dashboard

## Setup Instructions
1. Clone or navigate to the project directory.
2. Install dependencies for frontend and backend.
3. Set up MongoDB database.
4. Run the backend server.
5. Run the frontend development server.

## Project Structure
- `/frontend`: React application
- `/backend`: Express server
- `/database`: MongoDB models and connection
